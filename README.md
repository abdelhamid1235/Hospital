## 📖 About The Project

This is a static front-end medical website designed to present hospital
information in a professional way.

The goal of this project was to:

-   Practice semantic HTML structure\
-   Improve CSS layout skills\
-   Build a responsive multi-section website\
-   Create a clean and modern UI design

------------------------------------------------------------------------
